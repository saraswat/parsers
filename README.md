# parsers
Links to interesting English parsers

## Frame-based parser
From Noah Smith [SEMAFOR](http://demo.ark.cs.cmu.edu/parse). Produces instantiated frames, from [Frame-net](https://framenet.icsi.berkeley.edu/fndrupal/about).

## CandC parser
Web access to the Clark and Curran parser is [available here](http://valeriobasile.github.io/candcapi/)
with the [source code here](https://github.com/valeriobasile/candcapi).

## CCG based parser
Mike Lewis' [EasySRL](https://github.com/mikelewis0/EasySRL). Produces lambda forms.

ALso the underlying [EasyCCG parser](http://homepages.inf.ed.ac.uk/s1049478/easyccg.html).

## Google's SyntaxNet parser
[SyntaxNet](https://github.com/tensorflow/models/tree/master/syntaxnet)
Based on the Nivre parser.

## Stanford NLP
[Stanford](http://nlp.stanford.edu/software/)

## ACE
[Attempto Parsing Engine](https://github.com/Attempto/APE)
